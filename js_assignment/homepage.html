About

HTML TAG GENERATOR is a user-friendly site which helps users with little knowldege about html and it's syntax to easily identify and use them for an future applications. It's designed using HTML, CSS and JAVASCRIPT. The reason behind it's simplicity is that the user is just required to click on the element that he wants to insert.
The tag with it's default attribites are printed onto the screen. This piece of code can be copied and pasted into a much larger fragment bit by bit and a fully fledged website can be created.
